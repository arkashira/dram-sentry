# Breakeven Analysis – dram‑sentry

| Metric | Value |
|--------|-------|
| **Unit Cost per Active User** | **$0.035 /mo** (Compute $0.020 + Storage $0.010 + Bandwidth $0.005) |
| **Fixed Monthly Operating Cost** | **$50,000** (dev, infra, support, marketing) |
| **Gross Margin** | **70 %** (Revenue – Variable Cost) |

## Pricing Tiers

| Tier | Price /mo | Core Features |
|------|-----------|---------------|
| **Basic** | **$5** | Real‑time price feed, 1‑day forecast, email alerts |
| **Standard** | **$15** | All Basic + 7‑day forecast, SMS alerts, API access (10 k calls/mo) |
| **Premium** | **$35** | All Standard + 30‑day forecast, Webhooks, 100 k API calls/mo, dedicated account manager |

## Customer Acquisition Cost (CAC)

| Channel | CAC Range |
|---------|-----------|
| Direct sales + partner channel | **$200 – $400** |
| In‑market digital ads | **$150 – $300** |
| Channel partners (OEMs) | **$250 – $350** |

## Lifetime Value (LTV)

Assuming 12 mo average tenure and 70 % gross margin:

| Tier | LTV |
|------|-----|
| Basic | $5 × 12 × 0.70 = **$42** |
| Standard | $15 × 12 × 0.70 = **$126** |
| Premium | $35 × 12 × 0.70 = **$294** |

## Break‑Even User Count

Break‑even MRR = Fixed Costs / Gross Margin  
`$50,000 / 0.70 ≈ $71,429` per month.

| Tier | Users Needed to Reach $71,429 MRR |
|------|-----------------------------------|
| Basic | $71,429 / $5 ≈ **14,286** users |
| Standard | $71,429 / $15 ≈ **4,762** users |
| Premium | $71,429 / $35 ≈ **2,041** users |

## Path to $10 k MRR

| Target MRR | Tier | Users Required | Notes |
|------------|------|----------------|-------|
| $10 k | Basic | 2,000 users | 2,000 × $5 = $10 k |
| $10 k | Standard | 667 users | 667 × $15 = $10 k |
| $10 k | Premium | 286 users | 286 × $35 = $10 k |

**Recommended Ramp‑Up Plan**

1. **Launch Basic** – Acquire 2,000 users in 3 months via OEM channel → $10 k MRR.  
2. **Upsell to Standard** – Convert 30 % of Basic users → +$3 k MRR.  
3. **Introduce Premium** – Target 200 users in 6 months → +$7 k MRR.  

Total: **$20 k MRR** after 6 months, comfortably above the $71 k break‑even threshold when combined with projected growth in fixed costs.
